## Descripción

Repositorio con varios ejemplos prácticos  utilizando **Spring Boot**. Cada ejemplo está diseñado para demostrar funcionalidades específicas del framework, desde conceptos básicos hasta integraciones avanzadas.

---

## Tecnologías/Fundamentos



- **Java** 
- **Spring Boot** 
- **JDBC** - (Java Database Connectivity) - El estándar base
- **JPA** - Define un conjunto de reglas y estándares para el mapeo objeto-relacional
- **Hibernate** - El ORM más popular
- **Hikari** - Connection pool más rápido
- **H2 Database** - Base de datos en memoria para pruebas y desarrollo
- **PostgreSQL** 
- **Maven** - Herramienta de gestión y construcción de proyectos
- **JWT**- JSON Web Token
